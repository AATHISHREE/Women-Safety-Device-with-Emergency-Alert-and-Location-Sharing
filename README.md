# Women-Safety-Device-with-Emergency-Alert-and-Location-Sharing

🔹 Description

This project presents a smart embedded system designed to enhance women safety. 
It uses sensors to detect potential threats and provides immediate alert response. 
The system is designed to be extended with GPS and GSM modules for emergency location sharing.

🔹 Features

- Real-time threat detection
- Ultrasonic distance sensing
- IR motion detection
- Automatic alert system (LED + Servo)
- Scalable for GPS and GSM integration
  
🔹 Components Used

- ESP32
- Ultrasonic Sensor
- IR Sensor
- Servo Motor
- LED
- Breadboard
  
🔹 Working Principle

The system continuously monitors surroundings using sensors. 
When a threat is detected, the ESP32 triggers an alert. 
Future enhancement includes sending location via GPS and GSM.

🔹 Future Scope

- GPS-based location tracking
- GSM-based SMS alerts
- Mobile app integration
- Wearable device implementation
